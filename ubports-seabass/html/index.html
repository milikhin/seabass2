Seabass for UBports v0.5.0

Release notes

v0.5 adds ability to build projects with Clickable. To build a project open clickable.json and click on the "Build" icon.

Please note that this feature is very-very-very experimental. There are a few requirements:

  • the project files should be located within Downloads or Documents directory
  • your device should have Libertine command line tools (preinstalled with the operating system)

Your feedback on github or in the dedicated topic on the UBports forum is very appreciated!
